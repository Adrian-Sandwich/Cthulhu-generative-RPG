# 🏮 The Lighthouse - Web Interface

Modern HTML/CSS/JavaScript web interface for the Cthulhu generative RPG.

## Quick Start

### 1. Install Dependencies

```bash
cd /Users/adrianmedina/src/Cthulhu
pip install flask flask-cors
```

### 2. Run the Server

```bash
python3 app.py
```

You should see:
```
 * Running on http://127.0.0.1:5000
```

### 3. Open in Browser

Navigate to: **http://localhost:5000**

## Features

### Game Interface
- **Character Stats Panel** - Real-time HP/SAN/Luck tracking with visual bars
- **Location Display** - Current game location
- **Turn Counter** - Track gameplay progression
- **Scene Display** - ASCII art environments rendered in monospace font
- **Narrative Log** - Full turn-by-turn game history
- **Action Input** - Submit investigator actions

### Lovecraftian Theme
- Dark color scheme (primary: #0a0e27)
- Accent lighting (#c0a080 - aged gold)
- Serif typography for immersion
- Monospace fonts for ASCII art
- Atmospheric UI with subtle shadows and gradients

### Gameplay
1. **Create Character**
   - Choose investigator name
   - Select archetype (Scholar, Detective, Occultist, Wanderer)
   
2. **Explore**
   - Scenes display atmospheric ASCII art
   - Take actions using natural language
   - LLM generates narrative responses
   
3. **Track Progress**
   - View game statistics in real-time
   - Review full history anytime
   - See turn-by-turn progression

## API Endpoints

All endpoints are RESTful JSON APIs.

### Game Management
- `POST /api/game/start` - Initialize new game
- `GET /api/game/state` - Get current game state
- `POST /api/game/action` - Process player action
- `POST /api/game/reset` - Reset game

### Scene Management
- `GET /api/scenes` - List available scenes
- `GET /api/scene/<key>` - Get specific scene

## Architecture

### Backend (Flask)
- `app.py` - Flask server, API endpoints
- Integrates with `GenerativeGameEngine`
- Manages game state and narrative

### Frontend (HTML/CSS/JS)
- `templates/index.html` - Main interface markup
- `static/style.css` - Lovecraftian theme styling
- `static/app.js` - Client-side logic and API calls

### Game Engine
- `core/game_generative.py` - LLM-based narrative generation
- `graphics_engine/ascii_scenes_hd.py` - ASCII art scenes
- `core/game_state.py` - Game state management

## Customization

### Colors
Edit `:root` variables in `static/style.css`:
```css
--primary-dark: #0a0e27;
--accent-light: #c0a080;
--text-mono: #d4af37;
```

### Fonts
Change in `static/style.css`:
```css
--mono-font: 'Courier New', 'Courier', monospace;
--title-font: 'Georgia', serif;
```

### Scene Integration
Currently shows `exterior_storm` by default. To make scenes dynamic:
1. Have `GenerativeGameEngine` track current location
2. Map location to scene key in `app.py`
3. Update frontend to fetch appropriate scene

## Keyboard Shortcuts

- **Type** - Focus action input automatically
- **Enter** - Submit action
- **View Scene** - Show current location ASCII art
- **Show History** - View all turns played
- **New Game** - Reset and start over

## Troubleshooting

### Port Already in Use
```bash
# Use different port
python3 app.py --port 5001
```

### Module Not Found
```bash
pip install -r requirements.txt
```

### Font Not Rendering
- Check browser supports monospace fonts
- Try refreshing (Ctrl+Shift+R / Cmd+Shift+R)
