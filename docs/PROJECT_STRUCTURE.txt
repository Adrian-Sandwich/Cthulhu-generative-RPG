================================================================================

CORE ENGINE:
  game_universal.py
    → Universal game motor (works with ANY entries_*.json)
    → Validate, play interactive, play automatic
    → Classes: UniversalGameEngine, Investigator, GameState, RollResult
    → ~250 lines, modular and extensible

ENHANCED FEATURES:
  game_enhanced.py
    → Extended motor with advanced features
    → Save/Load system with JSON persistence
    → Journal with event logging and timestamps
    → Better state tracking (SAN, Luck, HP, MP)
    → Classes: Journal, GameSave, EnhancedGameEngine

USER INTERFACE:
  game_with_investigators.py
    → Interactive menu for investigator selection
    → Display full character sheets
    → Loads real investigator data from investigators.json
    → Ready for game flow integration
